# CoCタイトルbot 操作ガイド
このBotは、クトゥルフ神話TRPG（CoC）のセッション情報を、決められたフォーマットで自動投稿するツールです。

## 使いかた
チャット欄に` /coct` と入力してください。

## 入力する項目
コマンドを入力すると以下の項目が出てくるので、順番に入力してください。

- version ：CoCの版数（6 か 7）を入力してください。

- gm ：GMの名前を入力してください。

- count ：その卓の回数を入力してください。

- title ：シナリオのタイトルを入力してください。

- url ：ココフォリアなどの部屋URLを入力してください。

## 出力される形
入力が終わると、Botが以下のように投稿してくれます。

入力例:
`version: 7, gm: 山田, count: 1, title: 深淵の呼び声, url: https://...`

出力結果:
```
【新 CoC シナリオ】深淵の呼び声【山田卓 1】
https://...
```
※ version に 7 を入れると「新 CoC シナリオ」、それ以外（6など）を入れると「CoC シナリオ」と自動で判別されます。
